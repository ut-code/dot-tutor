## Development

### Requirements

- [npm](https://github.com/npm/cli)

### Installation

```shell
cd learn-braille
npm ci
```

### Local Development

```shell
npm run dev
```

After running the command above, visit [http://localhost:3000/](http://localhost:3000/).
