## 開発

### 要件

- [npm](https://github.com/npm/cli)

### 環境構築

```shell
cd learn-braille
npm ci
```

### 開発

```shell
npm run dev
```

上記のコマンドを実行した後、[http://localhost:3000/](http://localhost:3000/) にアクセスしてください。
