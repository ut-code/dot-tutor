### 漢点字を「部首」からみてみよう①

前項で習ったことを生かして、少し発展的な内容を学習します。
漢点字は、単なる点の羅列ではなく、一定の法則性を持って成り立っています。
１マスの漢点字は偏（へん）や旁（つくり）、冠や構えなどの「部首」にもなって、２マスまたは３マスの漢点字を作ります。

#### 具体例（き・きへん）

| き      | きへん   |          |          | 例外     |          |
| ------- | -------- | -------- | -------- | -------- | -------- |
| 「木」⢏ | 「校」⢇⢎ | 「林」⢇⢎ | 「杖」⢇⣸ | 「桃」⢥⠬ | 「松」⢥⢜ |

共通の「⢇」という構造を持っています。それは、「⢆」が「木」の意味を表しており、

- 「木（⢏）」 = 始点 + 「⢆」 + 終点

- 「きへん（⢇）」 = 始点 +「⢆」

となるからです（ただし例外が多い）。
