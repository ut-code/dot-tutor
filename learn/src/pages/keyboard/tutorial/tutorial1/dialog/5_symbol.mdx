## 記号の入力について

- 句点 ⠲
  - 「。」に対応する。後ろに空白を 2 つ置いて使用する。
  - ピリオドとして用いることもできる。この場合、後ろに空白を 1 つ置く。
- 読点 ⠰
  - 「、」「,」に対応。
- 中点 ⠐
  - 対等な関係で並ぶ語句の区切りとして用いられる「・」に対応させて用いる。後ろに空白を一つ置く。他の用途での「・」は省略または空白で表現する。
- 疑問符 ⠢ 感嘆符 ⠖
  - 文末のときは後ろに 2 つ空白を置き、文中のときは後ろに 1 つ空白を置く。カギ類、カッコ類の閉じ符号が続く場合は空白を置かない。
- 星印(第一星印「⠀⠀⠔⠔⠀」第二星印「⠀⠀⠢⠢⠀」第三星印「⠠⠢⠀」)
  - 主として第１星印を用いる。第１星印や第２星印は行頭にのみ用い、行頭を２マスあける。第３星印は文中で強調したい箇所の前に用いることができる。
- 囲みの記号
  - カギ類
    - 第一カギ「⠤ ⠤」、ふたえカギ「⠰⠤⠤⠆」は「」、『』にほぼ対応させる。
    - 第二カギ「⠰⠄ ⠠⠆」は””、＜＞などに対応させる。
    - 内側は続けて書き、外側は分かち書きの規則に従う。
  - カッコ類
    - 語句や文の直後に注記をする必要がある場合には、第１カッコ 「⠶ ⠶」 を用いる。第１カッコの中でさらにカッコを必要とする場合には、二重カッコ「⠰⠶ ⠶⠆」を用いる。
    - 第２カッコ「⠐⠶ ⠶⠂」は〔 〕などに対応させる。
    - 内側は続けて書き、外側は分かち書きの規則に従う。
  - 指示符類
    - 文の語句や一部を強調したり指定したりする場合に用いる。一般書では第１指示符「⠐⠤ ⠤⠂」を用いる。
- 線類
  - 傍線 「 ⠒⠒ 」
    - 前後を関係付けたり，対等な関係を示す意味をもっている。
  - 点線 「 ⠂⠂⠂ 」
  - 矢印 → 「 ⠒⠒⠕ 」、 ← 「 ⠪⠒⠒ 」、↔ 「 ⠪⠒⠒⠕ 」
    - 必要に応じて棒の長さを増減させてもよい。
  - 波線 「⠤⠤」
    - 文の内容が範囲を指定する場合は波線を用いる。
