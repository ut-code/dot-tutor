### 五十音の点字を読んでみよう！

五十音（「あ」から「ん」）は次の表に従います。

このように右下の 3 つの点が子音に対応し、母音との組み合わせで 50 音を表します。

また、「ん」「ー」(伸ばす棒)「っ」もそれぞれ下のように表されます。

| 五十音  |         |         |         |         |
| ------- | ------- | ------- | ------- | ------- |
| 「あ」⠁ | 「い」⠃ | 「う」⠉ | 「え」⠋ | 「お」⠊ |
| 「か」⠡ | 「き」⠣ | 「く」⠩ | 「け」⠫ | 「こ」⠪ |
| 「さ」⠱ | 「し」⠳ | 「す」⠹ | 「せ」⠻ | 「そ」⠺ |
| 「た」⠕ | 「ち」⠗ | 「つ」⠝ | 「て」⠟ | 「と」⠞ |
| 「な」⠅ | 「に」⠇ | 「ぬ」⠍ | 「ね」⠏ | 「の」⠎ |
| 「は」⠥ | 「ひ」⠧ | 「ふ」⠭ | 「へ」⠯ | 「ほ」⠮ |
| 「ま」⠵ | 「み」⠷ | 「む」⠽ | 「め」⠿ | 「も」⠾ |
| 「や」⠌ |         | 「ゆ」⠬ |         | 「よ」⠜ |
| 「ら」⠑ | 「り」⠓ | 「る」⠙ | 「れ」⠛ | 「ろ」⠚ |
| 「わ」⠄ |         | 「を」⠔ |         | 「ん」⠴ |
| 「ー」⠒ |         | 「っ」⠂ |         |         |
