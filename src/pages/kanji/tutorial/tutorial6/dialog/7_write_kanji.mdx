### 漢点字の入力

漢点字は、その名の通り、漢字を点字で表したものです。次のような 1 マスから 3 マスの 8 点式点字の列で、1 つの漢字が表現されます。
例えば、
|具体例||
| -------- | -------- |
| 「一」⢡⠊ | 「木」⢏ |

#### 漢点字の成り立ち

漢点字は、6 点の組み合わせの上に、漢字の印（始点・終点）をつけたものです。
例えば、1 マスの点字の場合、
|1 マスの点字|
| -------- |
| 「貝」⡩ |

いちばん上の 2 つの点が、この点字が一文字の漢字であることを表し、下の 6 点はその漢字の意味を表します。
2 マス、3 マスの点字でも同様に、
|2 マスの点字|
| -------- |
| 「三」⢡⠚ |

1 マスめの左上の点と、最後のマスの右上の点とが一文字の漢字の区切りを表し、その他の部分が漢字の意味を表します。
