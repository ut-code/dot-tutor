### 漢点字の仕組み

前項で習ったことを生かして、少し発展的な内容を学習します。
漢点字は、単なる点の羅列ではなく、一定の法則性を持って成り立っています。
１マスの漢点字は偏（へん）や旁（つくり）、冠や構えなどの「部首」にもなって、２マスまたは３マスの漢点字を作ります。

#### 具体例

| ① き    | きへん   |          |          | 例外     |          |
| ------- | -------- | -------- | -------- | -------- | -------- |
| 「木」⢏ | 「校」⢇⢎ | 「林」⢇⢎ | 「杖」⢇⣸ | 「桃」⢥⠬ | 「松」⢥⢜ |

共通の「⢇」という構造を持っています。それは、「⢆」が「木」の意味を表しており、

- 「木（⢏）」 = 始点 + 「⢆」 + 終点

- 「きへん（⢇）」 = 始点 +「⢆」

となるからです（ただし例外が多い）。

| ② みず  | さんずい |          |          |
| ------- | -------- | -------- | -------- |
| 「水」⡏ | 「海」⡇⣊ | 「河」⡇⢊ | 「湖」⡇⠪ |

共通の「⡇」という構造を持っています。

| ③ ひと  | にんべん |          |          | 例外     |          |
| ------- | -------- | -------- | -------- | -------- | -------- |
| 「人」⡋ | 「仲」⡃⠬ | 「休」⡃⢎ | 「付」⡃⢮ | 「代」⢑⣬ | 「任」⢑⡎ |

共通の「⡃」という構造を持っています。（ただしこれも例外が多い）。

| ④ しんにょう |          |          |
| ------------ | -------- | -------- |
| 「道」⣇⠘     | 「通」⣃⡺ | 「送」⣇⢞ |

共通の「⣇」という構造を持っています。

| ⑤ もん  | もんがまえ |          |          |
| ------- | ---------- | -------- | -------- |
| 「門」⣽ | 「問」⣵⠾   | 「関」⣵⢞ | 「間」⣵⣌ |

もんがまえをもつ漢字は「⣵」を共通に持ちます。
