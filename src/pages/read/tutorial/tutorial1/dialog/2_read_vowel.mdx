### 母音を読んでみよう

母音（あ～お）は次の表に従って表現されます。

このように母音では左上の 3 つの点が使われます。

| 母音    |         |         |         |         |
| ------- | ------- | ------- | ------- | ------- |
| 「ア」⠁ | 「イ」⠃ | 「ウ」⠉ | 「エ」⠋ | 「オ」⠊ |

### 練習しよう！

次の点字を読んで、ひらがなで解答を入力してみましょう。

入力したら、「答え合わせ」のボタンを押して、自分の解答が正しいか確認してください！
