### 濁音、半濁音を読んでみよう！

濁音、半濁音は次の表に従います。

"⠐"は後に続く文字が濁音、"⠠"は後に続く文字が半濁音であることを示します。

前のセクションで習った五十音に"⠐"か"⠠"を付けるだけで、濁音半濁音が表せます！

| 濁音     |          |          |          |          |
| -------- | -------- | -------- | -------- | -------- |
| 「が」⠐⠡ | 「ぎ」⠐⠣ | 「ぐ」⠐⠩ | 「げ」⠐⠫ | 「ご」⠐⠪ |
| 「ざ」⠐⠱ | 「じ」⠐⠳ | 「ず」⠐⠹ | 「ぜ」⠐⠻ | 「ぞ」⠐⠺ |
| 「だ」⠐⠕ | 「ぢ」⠐⠗ | 「づ」⠐⠝ | 「で」⠐⠟ | 「ど」⠐⠞ |
| 「ば」⠐⠥ | 「び」⠐⠧ | 「ぶ」⠐⠭ | 「べ」⠐⠯ | 「ぼ」⠐⠮ |

| 半濁音   |          |          |          |          |
| -------- | -------- | -------- | -------- | -------- |
| 「ぱ」⠠⠥ | 「ぴ」⠠⠧ | 「ぷ」⠠⠭ | 「ぺ」⠠⠯ | 「ぽ」⠠⠮ |
