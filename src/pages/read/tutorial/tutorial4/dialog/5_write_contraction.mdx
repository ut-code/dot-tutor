### 拗音(「ゃ」「ゅ」「ょ」)の入力について

拗音は次の表に従って入力していくことができます。

"⠈"は後に続く文字が拗音であることを示します。

その後に「か」を表す"⠡"が来れば「きゃ」、「く」を表す"⠡"が来れば「きゅ」、「こ」を表す"⠡"が来れば「きょ」になります。

| 拗音       |     |            |     |            |
| ---------- | --- | ---------- | --- | ---------- |
| 「きゃ」⠈⠡ |     | 「きゅ」⠈⠩ |     | 「きょ」⠈⠪ |
| 「しゃ」⠈⠱ |     | 「しゅ」⠈⠹ |     | 「しょ」⠈⠺ |
| 「ちゃ」⠈⠕ |     | 「ちゅ」⠈⠝ |     | 「ちょ」⠈⠞ |
| 「ひゃ」⠈⠥ |     | 「ひゅ」⠈⠭ |     | 「ひょ」⠈⠮ |
| 「みゃ」⠈⠵ |     | 「みゅ」⠈⠽ |     | 「みょ」⠈⠾ |
| 「りゃ」⠈⠑ |     | 「りゅ」⠈⠙ |     | 「りょ」⠈⠚ |

### 拗濁音と拗半濁音(「ぎゃ」「ぎゅ」「ぎょ」など)の入力について

拗濁音と拗半濁音は次の表に従って入力していくことができます。

拗音で用いた"⠈"の代わりに、拗濁音の場合は"⠘"、拗半濁音"⠨"を最初につけます。

| 拗濁音     |     |            |     |            |
| ---------- | --- | ---------- | --- | ---------- |
| 「ぎゃ」⠘⠡ |     | 「ぎゅ」⠘⠩ |     | 「ぎょ」⠘⠪ |
| 「じゃ」⠘⠱ |     | 「じゅ」⠘⠹ |     | 「じょ」⠘⠺ |
| 「ぢゃ」⠘⠕ |     | 「ぢゅ」⠘⠝ |     | 「ぢょ」⠘⠞ |
| 「びゃ」⠨⠥ |     | 「びゅ」⠨⠭ |     | 「びょ」⠨⠮ |
