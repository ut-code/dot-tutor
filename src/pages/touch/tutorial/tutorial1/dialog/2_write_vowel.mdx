### 母音の入力について

母音（あ～お）は次の表に従って入力していくことができます。

このように左上の 3 つの点を使って母音を表現します。

| 母音    |         |         |         |         |
| ------- | ------- | ------- | ------- | ------- |
| 「ア」⠁ | 「イ」⠃ | 「ウ」⠉ | 「エ」⠋ | 「オ」⠊ |

### 練習しよう！

<PracticeTouch
  question="Q1 「いえ」を入力してください。"
  answer="いえ"
  length={2}
/>
<br />
<PracticeTouch
  question="Q2 「あう」を入力してください。"
  answer="あう"
  length={2}
/>
<br />
<PracticeTouch
  question="Q2 「あおい」を入力してください。"
  answer="あおい"
  length={3}
/>
